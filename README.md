## Hagura-Academic

Hagura-Academic is a responsive, dark-mode ready theme made for academia. It is based on [Hagura](http://webjeda.com/hagura) developed by [WebJeda](http://webjeda.com)

## Features
- Responsive
- Dark Mode Toggle
- Project Page
