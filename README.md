# Gradfolio

Responsive, dark-mode ready theme made for academia.

## Features
- Responsive
- Respects Dark Mode preference set by user
- Project Showcase

## Installation
* Fork the repository
* Your new site should be ready at https://username.github.io/gradfolio/
* Change content and add custom domain
